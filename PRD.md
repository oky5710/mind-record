# Mind Profiler — PRD

## 개요

정신건강을 기록하는 개인용 앱. 정신과 진료(약 복용, 검사 결과)와 일상 컨디션(기분, 수면, 운동, 커피, 스트레스 이벤트)을 한곳에 기록하고, 시계열로 함께 보면서 패턴/인과관계를 파악해 정신과 진료 시 참고 자료로 쓰는 것이 목적. 
Google 로그인으로 사용자별 데이터 분리는 되어 있음.

## 화면 / 라우팅

| 경로 | 화면       | 설명 |
|---|----------|---|
| `/` | 진입 화면    | 고양이 사진 + 위로 문구, 기분/커피/복용약 간편 입력 |
| `/login` | 로그인      | Google 로그인 |
| `/calendar` | 달력보기     | 달력에서 날짜 선택 → 유형별 기록 입력/수정/삭제 |
| `/hrv-analysis` | 오늘의 패턴   | 일/시간/월 단위 HRV 차트, 수면·운동·구글 캘린더·커피·기분·이벤트 오버레이 |
| `/chart` | 나의 Trend | HRV/웨어러블/기분/커피 통합 대시보드 |

## 핵심 기능

### 진입 화면 (`/`)
* Pixabay 고양이 사진 + 랜덤 위로 문구
* 오늘 기분 미입력 시 이모지로 바로 선택
* 커피 한 잔 간편 입력 (오늘 마신 잔 수 배지 표시)
* 아침/취침 복용약 퀵버튼 — 해당 시간대 약 전체를 한 번에 복용 처리, 오늘 이미 눌렀으면 색깔 있는 체크 배지 표시

### 달력보기 (`/calendar`)
달력에서 날짜를 클릭하면 유형을 선택해서 기록:
* **검사**: 정신과 HRV 검사 결과 (Time/Frequency Domain 수치 전체)
* **운동**: 종류/시간/강도
* **커피**: 종류/시간/메모, 하루 여러 잔이면 개수로 표시
* **기분**: 1~5점 이모지
* **약복용**: 아침/취침/필요시 체크박스로 선택한 시간대의 약 전체를 그 날짜로 복용 처리
* **이벤트**: 약 변경/대인관계 문제/업무 스트레스/병원 진료/기타(직접 입력) + 시간 + 설명

날짜 칸에는 그날 기록된 항목들이 종류별로 표시되고, 탭하면 상세 확인 후 **수정/삭제** 가능 (모든 기록 유형 공통).

### 복용약 관리 (`/medicine`)
* 식약처 공공 API로 약 검색 → 이름/모양/색/제조사 확인 후 등록
* 복용 시간대(아침/점심/저녁/취침전/필요시) 지정

### 오늘의 패턴 (`/hrv-analysis`) / 나의 Trend (`/chart`)
* d3 + 순수 SVG로 구현한 커스텀 시계열 차트 (라이브러리 미사용)
* HRV(심박변이) 라인 차트 위에 검사 시점 SDNN(세모), 이벤트(유형별 아이콘), 커피(컵 아이콘, 하루 2잔↑ 빨강 강조), 기분(하트, 점수별 색) 오버레이
* 수면/운동/구글 캘린더 일정을 한 레인에 겹쳐서 표시 (색으로 구분), 수면은 5시간 미만이면 빨강 강조 + 막대에 수면시간 텍스트
* 구글 캘린더 일정은 마우스오버/탭 시 제목·시간·장소·참석자 툴팁
* 일/시간/월 단위 확대, 날짜 점프, 가로 스크롤 + 드래그

### 웨어러블 / 애플워치 연동
* `WearableData`(일별 요약: 수면시간·품질, 평균 심박수/HRV)와 `WearableSample`(타임스탬프별 원시 샘플, 주로 HRV)로 이원화
* 실시간 연동은 **iOS 단축어 자동화**로 처리 (Apple가 서드파티 웹앱에 HealthKit을 직접 열어주지 않음):
  * HRV: 단축어가 건강 데이터를 읽어 `/wearable-samples/bulk`로 전송, 서버가 문자열 숫자·NDJSON 등 단축어 특유의 직렬화 형태를 방어적으로 파싱
  * 수면: 시작/종료 시각만 보내면 서버가 수면 시간을 계산
* 운동 기록처럼 단축어로 자동화가 안 되는 항목은 애플 헬스 "모든 건강 데이터 내보내기" XML을 받아 1회성 스크립트로 업서트

### 구글 캘린더 연동
* Google Calendar 읽기 전용 스코프로 일정 조회 (DB 저장 없이 그때그때 조회만)
* 종일 일정은 제외(단, 휴가는 포함), 최신순 정렬, 무한 스크롤로 과거 방향 확장
